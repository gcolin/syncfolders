Why Sync Folders?

You have some documents in a folder. You use it. You change it. And you want to backup these data easily. This simple tool do this task automatically. It backups the data of your source folder to your destination folder.

  • You add a file in the source folder, it copies it in your destination folder.
  • You update a file in the source folder, it overrides the old one in your destination folder.
  • You drop a file in the source folder, it deletes it in your destination folder.

How do I backup my folder to a folder located in another computer in my local network?

You need to mount the remote folder. Your remote folder will be in a virtual hard drive (Windows put Z:// by default). You can find how to do that in Internet.
