Pourquoi Sync Folders?

Vous avez des documents dans un dossier. Vous les utilisés. Vous les modifiés. Et vous souhaitez faire une sauvegarde de ces documents simplement. Cet outil fait cette tâche automatiquement en copiant les données de votre dossier source vers votre dossier de destination.

  • Vous ajoutez un ficher dans le dossier source, il est copié dans le dossier de destination.
  • Vous modifiez un ficher dans le dossier source, il est réécrit dans le dossier de destination.
  • Vous supprimez un ficher dans le dossier source, il est supprimé dans le dossier de destination.

Comment sauvegarder un dossier vers un dossier distant (situé dans un autre ordinateur de mon réseau local)?

Vous devez monter le dossier distant. Votre dossier distant va devenir un disque virtuel (Windows assigne Z:// par défaut). Vous trouverez plus d'informations sur le montage de dossier avec Internet.
